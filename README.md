# frontend

React SPA e-commerce frontend for the monitoring platform.

## Features

- Product browsing and search
- Shopping cart
- User authentication
- Order management
- Profile settings

## Tech Stack

- React 18
- Vite
- TypeScript
- TanStack Query
- React Router
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
